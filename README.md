# 聚高体育

  应用介绍:"聚高体育"app是一款预约棒球馆的应用。

  应用简介:预约棒球馆
  
    首页：明星场馆、新店开业、限时特惠和今日优选等棒球场馆；
    
    场馆：棒球相关场馆，场馆位置、营业时间、价格，一目了然；
    
    资讯：关于棒球的相关资讯，国内和国外均有；
    
    福利中心：用户签到可得积分，积分可直接兑换商品，在各个场馆均可兑换；
    
    我的：个人信息、我的预约、浏览记录、我的收藏、和相关设置。
